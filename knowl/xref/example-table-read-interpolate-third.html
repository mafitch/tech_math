Example 3.2.7.

When we want a value that is half way between two entries in a table, we can simply average them. However, if we want a value somewhere other than half way in between we must perform an additional calculation.
What is the stall speed with 10° bank angle with flaps in the Up setting?
The nearest entries in the table are 0° and 30°. We need to figure out what percent 10 is between 0 and 30. We can use that to calculate the matching number between the table entries (48 and 52). Percent is part/whole which in this case is
\begin{equation*} \frac{10}{30-0} = \frac{10}{30} = \frac{1}{3}\text{.} \end{equation*}
We do not need to write this as a percent (it would be approximately 33.3%), because we are just using it to multiply in the next step.
We want the speed that is 1/3 of the way between 48 and 52. Again we treat this as a percent problem. We want percent times the whole to calculate the part.
\begin{equation*} \frac{1}{3} \cdot (52-48) = \frac{4}{3}. \end{equation*}
This result is how far above 48, so the speed is
\begin{equation*} 48+4/3 \approx 49.3 \approx 50. \end{equation*}
Again we round up for safety.
Because there are only 4 knots between the entries, it hardly seems worthwhile to do this work, especially because we round up for safety. There are times however, when this process is useful.
