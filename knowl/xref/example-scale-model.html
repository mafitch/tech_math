Example 2.4.3. Scale Model.

A model of a space shuttle is labeled as 1:144. This means one inch on the model represents 144 inches on the actual shuttle. We can express this as a ratio between the size of objects on the model and the size of the objects on the actual shuttle. If we want to know how large a part of the shuttle is and we know on the model it is 1.72 inches we can set up the proportion
\begin{equation*} \frac{1 \text{ model inches}}{144 \text{ inch}} = \frac{1.72 \text{ model inches}}{s \text{ inch}}\text{.} \end{equation*}
